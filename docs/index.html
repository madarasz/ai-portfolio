Board Game Rules Bot

GitHub

Discord rules bot for Warhammer 40k: Kill Team. It can answer complex rules questions while providing rules citations and detailed logical explanations. It uses LLM + RAG to generate responses. It is LLM provider-agnostic and can integrate with models from OpenAI, Anthropic, Gemini, Grok, Mistral, DeepSeek and Kimi.

Features comprehensive unit tests, integration tests, retrieval tests, RAGAS-like LLM-as-a-judge evaluation, model comparison tests, and in-production observability.

  • Python
  • RAG
  • ChromaDB
  • Pydantic
  • LLM as a judge
  • Spec Kit

Tournament Tables

GitHub Website

A utility for organizing tournaments. It syncs with the Best Coast Pairings API for pairing data, then redistributes table assignments so every player plays on different terrain setups throughout the event. Dockerized environment for easy deployment. The design was made with Google Stitch. Unit tests and Playwright E2E tests for validation.

Datacard Manager

GitHub

Tooling for managing WH40k: Kill Team datacards. Automated processing pipeline for downloading PDF sources and extracting cards. It can export a custom set of cards to PDF for printing.

Shared OpenAPI 3.0 definitions as a contract between frontend and backend services. Extensive unit tests and Playwright E2E tests.
